To activate the virtual environment "venv" on Windows 11, use the following command in your command prompt or PowerShell:

For Command Prompt (cmd.exe):
venv\Scripts\activate.bat

For PowerShell:
venv\Scripts\Activate.ps1

Once activated, your command prompt will be prefixed with (venv) indicating the virtual environment is active.

To deactivate the virtual environment, simply run:
deactivate
